# Roadmap

## Current status

- Terminal UI with tactical grid view
- Seeded random map generation
- Pickup and hazard spawning
- Mobile WebSocket client prototype
- CI workflow for formatting, linting, and tests
- CLI help/version support

## Short-term goals

- [x] Implement WASD movement and controls
- [x] Support dynamic seed and map size selection
- [x] Add mobile WebSocket interface and browser client
- [x] Add firing actions and weapon mechanics
- [x] Implement mock serial fallback for hardware development
- [x] Improve in-game HUD and action feedback
- [x] Add integration tests for movement, action points, and pickups

## Medium-term goals

- [ ] Add enemy AI and turn-based combat
- [ ] Add sound effects or terminal vibration hints
- [ ] Add persistent player progress and scoreboards
- [ ] Build a full mobile web client with map rendering and status panel
- [ ] Support multiple remote clients and multiplayer state sharing

## Long-term goals

- [ ] Create a polished release with assets and documentation site
- [ ] Add a desktop GUI or native app wrapper
- [ ] Move to a proper release pipeline with GitHub Releases and changelog automation
- [ ] Add optional save/load and replay support
